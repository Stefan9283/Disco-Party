## Controls

| Mouse / Keyboard Keys           | Action                      |
| ------------------------------- | --------------------------- |
| Right Mouse Button + WASD       | move camera position        |
| Right Mouse Button + Mouse Drag | camera pan                  |
| Space                           | Change camera lighting mode |
